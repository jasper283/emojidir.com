# Emoji Directory에 오신 것을 환영합니다! 🎉

**Emoji Directory** 를 소개하게 되어 기쁩니다 - 여러 디자인 시스템의 이모지를 탐색하고 다운로드할 수 있는 원스톱 목적지입니다.

## 우리의 차별점

Emoji Directory의 특징:

### 🎨 여러 디자인 시스템
다음 플랫폼의 이모지를 탐색하세요:
- **[Microsoft Fluent Emoji](/ko/fluent-emoji)** - 현대적이고 표현력 있는 3D 디자인
- **[Google Noto Emoji](/ko/noto-emoji)** - 깨끗하고 미니멀한 디자인
- **[시스템 이모지](/ko/unicode-emoji)** - 기기에서 네이티브 Unicode 이모지로 표시

### 🔍 스마트 검색
지능형 검색 시스템은 다음을 지원합니다:
- 다국어(영어, 일본어, 한국어, 중국어)
- 키워드 및 의미 기반 검색
- 카테고리 필터링

### 💾 쉬운 다운로드
원하는 형식으로 이모지를 다운로드하세요:
- 일반 용도용 PNG
- 확장 가능한 그래픽용 SVG
- 원클릭 다운로드 - 등록 불필요

## 시작하기

1. **[탐색](/ko/fluent-emoji)** - 카테고리 또는 플랫폼별로 탐색
2. **[검색](/ko/fluent-emoji)** - 이름 또는 키워드로 특정 이모지 찾기
3. **다운로드** - 즉시 이모지 받기

## 완벽한 용도...

- 📱 **디자이너** - UI에 완벽한 이모지 찾기
- 🎨 **콘텐츠 크리에이터** - 시각적 콘텐츠 향상
- 💻 **개발자** - 애플리케이션용 이모지 액세스
- ✍️ **작가** - 글에서 더 많은 것을 표현

## 다음 단계는?

Emoji Directory를 지속적으로 개선 중:
- 더 많은 이모지 컬렉션
- 향상된 검색 기능
- 더 나은 사용자 경험
- 커뮤니티 기능

업데이트를 기대해 주세요!
