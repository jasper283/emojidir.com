# 歡迎來到 Emoji Directory！🎉

我們很高興向您介紹 **Emoji Directory** - 您探索和下載多個設計系統表情符號的一站式目的地。

## 我們的與眾不同之處

Emoji Directory 的突出特點包括：

### 🎨 多個設計系統
瀏覽來自以下平台的表情符號：
- **[Microsoft Fluent Emoji](/zh-TW/fluent-emoji)** - 現代、富有表現力的 3D 設計
- **[Google Noto Emoji](/zh-TW/noto-emoji)** - 簡潔、極簡的設計
- **[系統表情符號](/zh-TW/unicode-emoji)** - 在您的裝置上顯示為原生 Unicode 表情符號

### 🔍 智能搜尋
我們的智能搜尋系統支援：
- 多語言（英語、日語、韓語、中文）
- 基於關鍵詞和含義的搜尋
- 分類篩選

### 💾 輕鬆下載
以您喜歡的格式下載表情符號：
- PNG 格式用於通用用途
- SVG 格式用於可縮放圖形
- 一鍵下載 - 無需註冊

## 入門指南

1. **[瀏覽](/zh-TW/fluent-emoji)** - 按類別或平台探索
2. **[搜尋](/zh-TW/fluent-emoji)** - 按名稱或關鍵詞查找特定表情符號
3. **下載** - 即時獲取您的表情符號

## 完美適用於...

- 📱 **設計師** - 為您的 UI 找到完美的表情符號
- 🎨 **內容創作者** - 增強您的視覺內容
- 💻 **開發者** - 為您的應用程式訪問表情符號
- ✍️ **寫作者** - 在寫作中表達更多

- 增強的搜尋功能
- 更好的使用者體驗
