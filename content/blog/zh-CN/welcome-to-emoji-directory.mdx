# 欢迎来到 Emoji Directory！🎉

我们很高兴向您介绍 **Emoji Directory** - 您探索和下载多个设计系统表情符号的一站式目的地。

## 我们的与众不同之处

Emoji Directory 的突出特点包括：

### 🎨 多个设计系统
浏览来自以下平台的表情符号：
- **[Microsoft Fluent Emoji](/zh-CN/fluent-emoji)** - 现代、富有表现力的 3D 设计
- **[Google Noto Emoji](/zh-CN/noto-emoji)** - 简洁、极简的设计
- **[系统表情符号](/zh-CN/unicode-emoji)** - 在您的设备上显示为原生 Unicode 表情符号

### 🔍 智能搜索
我们的智能搜索系统支持：
- 多语言（英语、日语、韩语、中文）
- 基于关键词和含义的搜索
- 分类筛选

### 💾 轻松下载
以您喜欢的格式下载表情符号：
- PNG 格式用于通用用途
- SVG 格式用于可缩放图形
- 一键下载 - 无需注册

## 入门指南

1. **[浏览](/zh-CN/fluent-emoji)** - 按类别或平台探索
2. **[搜索](/zh-CN/fluent-emoji)** - 按名称或关键词查找特定表情符号
3. **下载** - 即时获取您的表情符号

## 完美适用于...

- 📱 **设计师** - 为您的 UI 找到完美的表情符号

- ✍️ **写作者** - 在写作中表达更多
